# To Do List

## 프로젝트 개요
이 프로젝트는 사용자가 할 일을 추가하고, 완료 상태를 체크하며, 필요시 삭제할 수 있는 **To Do List 애플리케이션**입니다. **React**와 **TypeScript**로 개발되었으며, 간결하고 직관적인 사용자 인터페이스와 함께 사용자 경험을 최우선으로 고려한 디자인이 특징입니다.

## 프로젝트 기간
- **시작일**: 2024-09-10
- **종료일**: 2024-09-25

## 주요 기능
- **할 일 추가**: 텍스트 입력 후 버튼을 눌러 새로운 할 일을 추가할 수 있습니다.
- **할 일 완료 표시**: 체크박스를 통해 완료한 할 일을 표시할 수 있습니다.
- **할 일 삭제**: 삭제 버튼을 눌러 불필요한 할 일을 제거할 수 있습니다.
- **직관적인 UI**: 간단하고 직관적인 디자인을 통해 사용성을 극대화하였습니다.

## 기술 스택
- **프론트엔드 프레임워크**: React
- **언어**: TypeScript, HTML, CSS
- **상태 관리**: React Hooks (useState)
- **배포**: GitHub Pages

## 리뷰 포인트
- **심플한 사용자 경험(UX)**: 사용자가 즉각적으로 기능을 이해하고 사용할 수 있는 심플한 구조입니다.
- **디자인 및 색감**: MZ 스타일을 반영하여 깔끔하면서도 현대적인 느낌을 줍니다.
- **코드 관리**: 코드가 컴포넌트별로 잘 분리되어 있어 유지보수에 용이합니다.
- **성능 최적화**: 소규모 프로젝트이지만, 성능 최적화에도 신경을 썼습니다.

## 부족한 점
- **모바일 최적화 미흡**: 현재 모바일 반응형 디자인이 부족하여 다양한 화면 크기에 대응할 수 없습니다.
- **기능 확장 필요**: 우선순위 설정, 기한 설정 등 추가적인 기능 구현이 필요합니다.
- **테스트 코드 미작성**: 유닛 테스트 및 통합 테스트가 미흡하여 향후 테스트 코드 추가가 필요합니다.
